# Qlik App to JSON (v2)

A mashup that allows you to export multiple Qlik apps to JSON format.

---

## 🔧 How to Install

### 📦 Qlik Sense Desktop

1. [Download the latest release](https://github.com/kekikcilerakin/qlik-app-to-json-v2/releases).
2. Unzip the contents into the following folder:
C:\Users[YourUsername]\Documents\Qlik\Sense\Extensions
3. Open Qlik Sense Desktop and access the mashup from the hub.

---

### 🌐 Qlik Sense Server (QMC)

1. Log in to the **Qlik Management Console (QMC)**.
2. Go to **"Extensions"**.
3. Click the **"Import"** button at the bottom of the page.
4. Select the mashup `.zip` file.
5. Click **"Import"** to upload.

---

## 🚀 Usage

- Open the mashup in your browser.
- Select one or more Qlik apps.
- Click **Export** to download them as JSON files.
